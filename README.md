```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

## Deploy with Streamlit Sharing / Render / Heroku

Quick steps to share via URL:

- Deploy on Render: create new Web Service, connect repo, set start command `streamlit run app.py`.
- Or use Streamlit Cloud: create app and point to `app.py`.
- Heroku: push branch and enable web dyno (Procfile included).
